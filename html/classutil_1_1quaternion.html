Detailed Description

template<class T>
class util::quaternion< T >

A class representing quaternions.

Quaternions serve to represent rotations in 3D, among other things.

Implicit conversion to the following template parameter types are allowed:

  • float
  • double
  • long double

◆ operator*()

template<class T >
quaternion< T > util::quaternion< T >::operator* ( const quaternion< T > &  a) const

Quaternion multiplication, i.e. the hamilton product.

Used primarily to combine two quaternions which represent rotations into a single quaternion which represents the result of applying both input quaternions in sequence. The operation is not commutative, and the second inout is the rotation that gets applied first.
