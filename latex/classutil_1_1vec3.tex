IMPORTANT\+: Operations between \doxylink{classutil_1_1vec3}{vec3} objects with different template parameters may return unexpected results. The right operand will be converted to the type of the left operand.

\item {\ttfamily signed long int}

\item {\ttfamily unsigned int}
\item {\ttfamily unsigned long int}
